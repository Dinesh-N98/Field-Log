# Marketing Executive Portal

- A responsive, mobile-first web application
- developed in React.js
- styled with Bootstrap
- communicating with a Spring Boot REST API.
- Field executives use it to log daily business activities including
    * customer calls
    * meetings conducted
    * leads generated
    * campaigns conducted.
- It includes an anonymous client financial categorization feature, which allows executives to evaluate and record customer financial capacity during interactions, protecting customer privacy and standardizing data entry.
